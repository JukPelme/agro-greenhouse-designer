## 2. Проектное решение (вариант failed_v1)

**Обоснование:** Участок площадью 500 м² в Новосибирской области — экстремально малый для целевой урожайности 2 000 т/год при возделывании томата круглогодично. Это физически непреодолимое ограничение: на данном участке невозможно разместить производственные мощности, которые обеспечат заявленный объём продукции, поэтому в варианте v3 выбрана предельно эффективная компоновка с максимальным использованием каждого квадратного метра. Выбран блочный многопролётный тип теплицы со стеклянным ограждением — стекло обеспечивает максимальное светопропускание, критичное для томата и сибирских зим с минимальной инсоляцией. Согласно требованию ТЗ, комплекс разделён на два отдельных блока для разделения культур по стадиям вегетации (рассадный и продуктивный отсеки). Высота в коньке принята достаточной для шпалерного выращивания томата в круглогодичном режиме. Грунт — глина с высоким УГВ (0,5 м), что требует свайного фундамента с периметральным дренажём и усиленной гидроизоляцией; это учтено в конструктивном решении. Подсобные зоны (технический узел, котельная, склад) размещены в пристройке для соответствия нормативной доле. Все инженерные сети (электро-, водо-, газоснабжение) отсутствуют на участке — необходимо автономное обеспечение, что дополнительно сокращает полезную площадь выращивания.

### ERROR — ENG.5-yield-feasibility

Площадь выращивания × норма культуры должна покрывать минимум 90% целевой урожайности из ТЗ. Иначе участок физически не вмещает заявленный объём продукции.

- Фактическое значение: `вмещает ~11 т/год (tomato @ 50.0 кг/м² × 216 м²)`
- Требуется: `>= 1800 т/год (90% от ТЗ 2000.0 т)`
- **Источник:** Инженерная проверка (не из СП): достижимость целевой урожайности
