## 2. Проектное решение (вариант v1)

**Обоснование:** Для круглогодичного выращивания томата в Краснодарском крае выбрано стекло (максимальное светопропускание 0.88, что критично при низкой зимней инсоляции 3.8 МДж/м²/день). Два одинаковых блока Venlo-типа размещены параллельно вдоль длинной стороны участка (200 м). Ширина блока 19.2 м (4 пролёта по 4.8 м), длина 192 м (кратно 6). Два блока занимают 2 × 192 × 19.2 = 7372.8 м² (~37% участка), что оставляет место для дороги между блоками, разрывов и подсобных зон. Высота в коньке 6.5 м и по водостоку 5.0 м обеспечивают шпалерную культуру томата и достаточный буферный объём воздуха. Целевая урожайность 500 т/год при площади 7372.8 м² даёт ~68 кг/м² — реалистично для современной стеклянной теплицы с досветкой и CO₂-подкормкой.
