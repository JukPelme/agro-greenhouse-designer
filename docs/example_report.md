## 2. Проектное решение (вариант v1)

**Обоснование:** Для круглогодичного выращивания томата в Краснодарском крае при целевой урожайности 25 кг/м²/год и участке 200×100 м выбрана классическая мультипролётная Venlo-теплица со стеклом (максимальное светопропускание 0.88, что критично зимой при низкой инсоляции 3.8 МДж/м²/сут). Два одинаковых блока по 6 пролётов 12.8 м (общая ширина 76.8 м с учётом стандарта Venlo 12.8 = 2×6.4) — для упрощения берём ширину 12 м из разрешённого ряда, по 6 блоков-пролётов конструктивно объединяем в один корпус шириной 72 м? Применяем разрешённый ряд: 2 блока шириной 25.6 м и длиной 192 м (кратно 6). Итого площадь блоков 2×25.6×192 = 9830 м² ≈ 49% участка (в пределах 75%). Высота в коньке 6.5 м, по водостоку 5.5 м — для шпалерной культуры томата с системой высокой подвязки. Подсобки размещаются в торце участка: энергоцентр (газа нет — электрокотельная + тепловой насос), склад субстрата/тары, упаковочный цех, бытовки. Целевая урожайность 500 т с 9830 м² = 50.9 кг/м²/год — реалистично для современной стеклянной теплицы с досветкой и CO₂-подкормкой (в индустрии 60–80 кг/м²). По участку остаются дороги, разрывы между блоками (мин. 6 м противопожарный), накопительные ёмкости для воды.

**Общая площадь под выращивание:** 9830 м²
