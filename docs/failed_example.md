## 2. Проектное решение (вариант v1)

**Обоснование:** Участок крайне мал (500 м²) для year_round томатного комплекса с целевой урожайностью 2000 т/год — это физически недостижимо (потребовалось бы ~40 т/м²/год при мировом рекорде ~100 кг/м²/год). Тем не менее, в рамках жёсткого ограничения участка 25×20 м предлагаю один стеклянный блок Venlo-типа максимально возможного размера с учётом отступов от границ (по 2 м с каждой стороны для пожарных проездов и инженерных коммуникаций). Блок 18×12 м (длина кратна 6, ширина из ряда), один пролёт 12 м, высота в коньке 7 м, по водостоку 6 м — для шпалерной культуры томата с досветкой. Стекло выбрано из-за сурового климата Новосибирска (СНР=2.4 кПа, t=-37°C) и максимального светопропускания при низкой зимней радиации (1.5 МДж/м²·сут). Подсобки вынесены в компактный модуль 4×6 м. ВНИМАНИЕ: данная компоновка заведомо не обеспечит 2000 т/год — это будет отражено валидатором; ТЗ требует пересмотра площади участка либо целевой урожайности. Дополнительные риски: отсутствие сетей (электричество, вода, газ), высокий УГВ (0.5 м) на глине — потребуется дренаж и автономные источники.
