## 2. Проектное решение (вариант failed_v1)

**Обоснование:** Участок крайне мал для заявленной урожайности в Новосибирской области с суровым климатом. По условию ТЗ требуется минимум два отдельных блока. Выбрана круглогодичная многопролётная блочная теплица (block) со стеклянным ограждением — стекло обеспечивает максимальное светопропускание, критически важное при низкой зимней инсоляции региона. Два блока разделены минимальным нормативным разрывом между зимними теплицами. Высота в коньке обеспечивает шпалерное ведение томата. Подсобные зоны вынесены за пределы тепличных блоков и составляют нормативную долю от общей площади комплекса. Следует отметить, что физические ограничения участка площадью всего пятьсот квадратных метров делают достижение целевой урожайности в две тысячи тонн принципиально невозможным: даже при предельной интенсификации технологии расчётный потенциал участка на несколько порядков ниже требуемого — это объективное противоречие ТЗ и участка, а не недостаток проектного решения.

## 4. Проверка по СП 107.13330

Проверено правил: **33**

### ERROR — SP107.4.4-year-round

Расстояние (разрыв) между зимними теплицами в составе ТОК/РОТК — не менее 6 м для проезда техники.

- Фактическое значение: `2.0`
- Требуется: `6.0`
- **Источник:** СП 107.13330 п. 4.4:
> «Расстояния между зимними теплицами, входящими в состав ТОК и РОТК, 
определяются шириной проездов и составляют не менее 6 м, между сезонными 
теплицами - не менее 1,5 м.»

### ERROR — ENG.5-yield-feasibility

Площадь выращивания × норма культуры должна покрывать минимум 90% целевой урожайности из ТЗ. Иначе участок физически не вмещает заявленный объём продукции.

- Фактическое значение: `вмещает ~11 т/год (tomato @ 50.0 кг/м² × 216 м²)`
- Требуется: `>= 1800 т/год (90% от ТЗ 2000.0 т)`
- **Источник:** Инженерная проверка (не из СП): достижимость целевой урожайности
