## 2. Проектное решение (вариант v1)

**Обоснование:** Для круглогодичного выращивания томата в Краснодарском крае выбрана блочная многопролётная зимняя теплица со стеклянным ограждением: стекло обеспечивает максимальное светопропускание, критичное для томата в зимний период при сравнительно невысокой инсоляции. Блочная компоновка из двух параллельных блоков оптимально вписывается в участок 200×100 м, позволяя разместить производственную площадь, необходимую для достижения целевой урожайности, при сохранении нормативного разрыва между блоками и периметрального проезда. Высота в коньке принята достаточной для шпалерного выращивания томата с подвязкой. Грунт — суглинок с глубиной залегания грунтовых вод более двух метров, поэтому принят ленточный фундамент глубиной около одного метра с гидроизоляцией; дренаж не требуется. Расстояние до животноводческих объектов значительно превышает нормативное, ограждение территории принято высотой 1,8 м.
