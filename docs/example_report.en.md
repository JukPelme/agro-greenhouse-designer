## 2. Design solution (variant v1)

**Rationale:** Для круглогодичного выращивания томата в Краснодарском крае выбрана блочная многопролётная теплица с остеклением: стекло обеспечивает максимальное светопропускание и долговечность, что критично для шпалерной культуры с подвязкой. Компоновка из двух крупных блоков позволяет вписаться в участок по длине и ширине, сохранив нормативный разрыв между блоками. Прямолинейный скат кровли с достаточным уклоном обеспечивает эффективный сток осадков и снеговую нагрузку по нормам. Высота в коньке принята с запасом сверх минимальных 5,5 м, необходимых для шпалерного ведения томата при круглогодичной эксплуатации. Подсобные зоны (котельная, склад, технический узел) размещены вдоль торца участка, не занимая тепличную площадь.
